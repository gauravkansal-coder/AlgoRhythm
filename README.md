To make the music react to your code, you need to run the "Spy" script on your computer.

### 1. Clone the Repo
```bash
git clone [https://github.com/gauravkansal-coder/algorhythm.git](https://github.com/gauravkansal-coder/algorhythm.git)

2. Run the Remote Spy
This script detects your typing activity and sends it to the cloud dashboard.

Bash
node remote-spy.js

3. Start Coding
1. Open the Live Dashboard link in your browser.

3. Open VS Code in the project folder.

4. Edit any file inside the app or components folder.(you can try adding a comment).

5. The spy specifically watches these folders for code changes.

6. Save your file (Ctrl + S) to trigger the signal.For a better experience,turn on auto save on vscode
